# Local Raw-Data Boundary

This directory is intentionally empty in the public release.

Raw official workbooks, filings and PDFs from PJM, Monitoring Analytics, MISO, FERC, SCC, ISO-NE and other operators/regulators are not redistributed. To rebuild beyond the released derived-data layer, download the official records listed in `../DATASETS_AND_LINKS.csv`, keep their original filenames where possible, and follow the page/table/cell locators in `../tables/Supplementary_Table_3_data_source_inventory.csv`.

The public smoke test and figure scripts use released derived inputs. Raw-source rebuilds are a local verification workflow and do not change the package boundary.
